# 🏛️ Máster en Ciencia de Datos y Máquinas de Aprendizaje con mención Inteligencia Artificial
## Trabajo de titulación **"Estudio comparativo de redes neuronales profundas para la predicción de tendencias de consumo energético en la provincia de Tungurahua."**

## 📖 Descripción

Este repositorio contiene el código fuente desarrollado para el trabajo de titulación enfocado en la predicción del consumo energético residencial en la provincia de Tungurahua, Ecuador, mediante técnicas de aprendizaje profundo (*Deep Learning*).

La investigación fue desarrollada siguiendo la metodología **CRISP-DM (Cross Industry Standard Process for Data Mining)**, la cual estructura el proyecto en fases de comprensión del problema, comprensión y preparación de los datos, modelado, evaluación e interpretación de resultados.

Como parte del estudio, se implementaron procesos de limpieza, transformación e imputación de datos, análisis exploratorio de datos (EDA), construcción de series temporales y evaluación de diferentes arquitecturas de redes neuronales para series temporales, incluyendo modelos **LSTM (Long Short-Term Memory)**, **GRU (Gated Recurrent Unit)** y **Transformer**.

El objetivo principal fue comparar el desempeño de estas arquitecturas y seleccionar el modelo con mejor capacidad predictiva para estimar el consumo energético mensual del sector residencial de la provincia de Tungurahua.

## 📄 Objetivo

Desarrollar y evaluar modelos predictivos capaces de estimar el consumo energético residencial mensual utilizando información histórica de consumo, variables climáticas y variables demográficas.
